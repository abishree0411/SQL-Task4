## Objective:
Use SQL aggregate functions and grouping to summarize and analyze tabular data.

| 2      | Bob     | IT         | 60000  |

| 5      | Eve     | Finance    | 58000  |
## SQL Queries:
1. Total salary of all employees
2. Average salary
3. Count of employees
4. Group salary by department
5. Group average salary by department
